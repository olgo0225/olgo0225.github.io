Olivia Golden

MAXAR Simulation Documentation

The summer between my sophomore and junior year of college I worked at an aerospace company called Maxar. I worked with the simulation team to document high-level Python code for wideband telemetry and monitored simulation instances. The code simulated retrieving encrypted telemetry data from a satellite, decrypting it, and relaying the information to the user. I had to form a deep understanding of the code in order to simply and concisely relay it to a user who might not be familiar with Python code.

Contrail Flight Search

My sophomore year of college I worked with a team of CU students to create a flight search software. A user is able to search for the cheapest flights by inputting their destination and departure locations as well as their travel dates. A user is also able to save results and view them later by logging into their account. This project utilized an API called Skyscanner.

Undergraduate Research, Department of Computer Science at the University of Colorado Boulder

My freshman year of college I worked as an Undergraduate researcher at CU Boulder. The project was to create a model that predicted the terrestrial water cycle at a local to global scale. I worked on verifying the results through validation and verification techniques. I also interpreted data and created interactive graphs with Python.
